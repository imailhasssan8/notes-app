         Notes App

A simple **Notes App** built using **HTML, CSS, and JavaScript**.
This project allows users to create, edit, and delete notes directly in the browser.

        Features

* Add new notes
* Edit existing notes
* Delete notes
* Simple and clean user interface
* Works directly in the browser

     Technologies Used

* HTML
* CSS
* JavaScript

    Run the Project

1. Download or clone this repository.
2. Open the project folder.
3. Run the **index.html** file in your browser.

        Purpose of the Project

This project was created to practice and understand the basics of **frontend web development**, especially working with JavaScript and DOM manipulation.
